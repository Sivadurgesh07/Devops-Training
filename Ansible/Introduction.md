"What is IAC?"
Creating Infrastructure Using Code - Definition

IAC can be divided in three categories :
 1. Infra Provisioning Tool
 2. Configuration Management Tool
 3. Server Templating Tool (Image Building Tool)

EXAMPLE:
INFRASTRUCTURE PROVISIONING -> Creating the EC2 instance,VPC,S3 are the examples of infrastructure provisiong. 
CONFIGURATION MANAGEMENT -> Installing necessary software inside the EC2 instance is the example of Configuration Management.
SERVER TEMPLATING TOOL -> Creating the AMI(Image) for the future use and other people to use it we have server templating tool.

Infra Provisioning Tools Example:
1. AWS CloudFormation
2. Terraform
3. Pulumi
4. Azure ARM Templates

Confiuration Management Tools Example:
1. Ansible 
2. Chef
3. Puppet
4. Saltstacks
